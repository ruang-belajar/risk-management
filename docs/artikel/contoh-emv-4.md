## 4️⃣ Kasus Proyek Konstruksi: Risiko Penundaan Izin

Sebuah proyek pembangunan gedung menghadapi risiko penundaan perizinan dari otoritas lokal. Penundaan ini akan berdampak pada biaya karena harus memperpanjang sewa alat berat dan menggaji tim inti lebih lama. Manajemen risiko mengidentifikasi tiga kemungkinan skenario:

### Data Kasus

| **Skenario Penundaan**                | **Probabilitas (P)** | **Dampak Moneter (I) (Biaya Tambahan/Kerugian)** |
| ------------------------------------- | -------------------- | ------------------------------------------------ |
| **Tidak Ada Penundaan**               | 65% (0.65)           | Rp0                                              |
| **Penundaan Singkat** (1-2 minggu)    | 25% (0.25)           | Rp60.000.000                                     |
| **Penundaan Panjang** ( $> 2$ minggu) | 10% (0.10)           | Rp180.000.000                                    |

### Perhitungan EMV

EMV dihitung untuk setiap skenario risiko dengan rumus: $EMV = P \times I$. Karena ini adalah ancaman (risiko negatif), dampak moneter (I) akan dianggap sebagai nilai negatif (biaya/kerugian).

1. EMV (Tidak Ada Penundaan):
    
    $$0.65 \times \text{Rp}0 = \text{Rp}0$$
    
2. EMV (Penundaan Singkat):
    
    $$0.25 \times (-\text{Rp}60.000.000) = -\text{Rp}15.000.000$$
    
3. EMV (Penundaan Panjang):
    
    $$0.10 \times (-\text{Rp}180.000.000) = -\text{Rp}18.000.000$$
    

### Hasil dan Kesimpulan

Total Expected Monetary Value (EMV):

$$\text{Total EMV} = \text{Rp}0 + (-\text{Rp}15.000.000) + (-\text{Rp}18.000.000) = \mathbf{-\text{Rp}33.000.000}$$

#### **Interpretasi**

Nilai **EMV sebesar -Rp33.000.000** menunjukkan bahwa, berdasarkan analisis probabilitas dan dampak yang ada, proyek tersebut **diharapkan** menanggung biaya tambahan (kerugian) sebesar **Rp33 Juta** akibat risiko penundaan izin.

Manajer Proyek harus menggunakan angka ini untuk:

1. **Mengalokasikan Kontingensi:** Memastikan setidaknya ada **Rp33 Juta** yang dimasukkan ke dalam anggaran kontingensi proyek untuk menanggapi risiko penundaan izin.
    
2. **Memprioritaskan Respon:** Angka ini menguatkan urgensi untuk segera menerapkan tindakan mitigasi, seperti bekerja sama lebih awal dengan otoritas perizinan atau menyewa konsultan khusus untuk mempercepat proses, demi mencegah biaya yang diharapkan ini menjadi nyata.
